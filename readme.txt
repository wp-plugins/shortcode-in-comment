== Description ==
This plugin enables blog owners to allow some shortcodes to be used in comments. You can choose what to allow (i.e. Whitelist some shortcodes). It only affects shortcodes in comments, but not in pages, posts, etc. All shortcodes are default to be disabled.

[Plugin Homepage](http://www.kelvinblog.tk/my-wordpress-plugins/shortcode-in-comment-page)

= Settings =
1. Visit 'Settings > Shortcode in Comment' to set what shortcodes are enabled in comments.
2. Type your shortcode(s) to be enabled in the text box (comma seperated for more than one)
3. Click 'Save Changes' to submit.

== Installation ==
= From Wordpress Dashboard =
1. Visit 'Plugins > Add New'.
2. Search for 'Shortcode in Comment'.
3. Click 'Install Now'.
4. Activate Shortcode in Comment from your Plugins page.
5. Visit 'Settings > Shortcode in Comment' to set what shortcodes are enabled in comments.

= Manually =
1. Download the zip-compressed plugin.
2. Extract and upload the folder 'shortcode-in-comment'.
3. Activate Shortcode in Comment from your Plugins page.
4. Visit 'Settings > Shortcode in Comment' to set what shortcode is enabled in comments.

== Frequently Asked Questions ==
= Will the plugin affect shortcodes in pages, posts, etc? =
No. It only affects shortcodes in comments.
